Source of Tales

This game started as an entry for the Liberated Pixel Cup 2012, where it was called 'Lurking Patrol Comrades'. It is a massive multiplayer online roleplaying game.

Starting the game

No releases are available yet so playing the game requires compiling at least the client from source. The client is based on Qt and is easy to compile on any operating system and can also be compiled for Android. See the wiki for instructions.
